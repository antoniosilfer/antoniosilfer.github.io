DESCRIPTION

Silfer File transfer is a powerful application to transfer files between devices (PC and Smart Phone). SFT is a 100% free application available yet for Windows and Android platform. /p>

SFT is a very simple application to use, requiring any user only 20 seconds to learn how to use the application.

Silfer file transfer always tries to take advantage of the maximum capacity of the devices thus enabling users to make transfers at high speeds often reaching tens of MB/s depending on the capabilities of the devices.

SFT does everything for the user, the only thing the SFT does not do is choose which devices to connect and select the files to the user since the SFT can not read human thoughts but provides the user with simple and robust features to complete these tasks in seconds.

FEATURES

  • Only application that supports multiple (sending and / or receiving) transfers between multiple devices. With SFT one device can be sending files (may be different) to multiple devices at the same time, and also that very instant the device may be receiving files from multiple devices.
  • A device can send and receive files to be the same device and at the same time. If a device is sending 10GB to another device, the other device also wants to send 100 MB, it is unnecessary to wait, just make another new connection and ready.
  • The SFT is the only one that allows a user to exchange text messages (Chat). This feature is very important when using the SFT in large enterprises, buildings or universities since with the SFT you can be easily connected with all the people around you. So when you need a file is no need to shout or go to the desired person to request the file. Send a message "I need a file AAA".
  • The SFT has the best tool to choose your files. With the file selector you can view the photos, play audio and video, and even search for any file. So you can tell the SFT that you want to see all the songs whose title contains the word "passion".
  • It gives the user the freedom always to choose the location where the files will be received. This feature is very important since all the user always want to have every type of file in a specific location thus avoiding mess.
  • Enables user to select files once and send to multiple devices simultaneously. Very important when you need to send the same files to multiple devices. Instead of selecting the same file 100 times to send 100 devices, selects 1 time and sends to 100 devices.
  • It supports transfer continuity at any time. Very important in cases of errors when a file already exists in receiver device or when a user have 300GB to send, so you send 150GB at a time and shall send the rest at another time, day, month, even year.
  • It provides relevant information transfer before the user accepts the transfer. It is always important to know how many files, the total size, and who sent the file before accepting the transfer.
  • Allows the user to send files over any network (WiFi, Ethernet, ...). Only on mobile networks that SFT does not allow for security reasons, but this feature may be enabled in future releases.
  • It allows the user to accept connections automatically or manually. This is because SFT is like a chat application, where users establish sessions and easily send messages (files) at any time.
  • It supports the sending of any kind and any file size.
